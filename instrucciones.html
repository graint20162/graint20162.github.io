Intrucciones

Objetivo

El objetivo de este juego es que tu mascota recoja las banderas y vaya luego contigo. Es una prueba de obediencia para él, por tanto, el juego solo termina si recojes todas las banderas.

¡USA LAS FLECHAS DERECHA E IZQUIERDA DE TU TECLADO Y LA BARRA ESPACIADORA PARA MOVERTE!. El entrenador no te recibirá con menos de 5 banderas.

Detalles del proyecto

Este juego fue realizado en Phaser. Esta es una librería utilizada para realizar animaciones para web que usa JavaScript. Puedes ver su funcionalidad aquí.


En la carpeta assets/js se encuentran tres archivos en JavaScript, llamados init.js. juego.js y fin.js. El primero es el que inicializa el juego, luego está el que presenta todos los elementos jugables, y el tercero se encarga de finalizar y reiniciar el juego. A estos tres archivos Phaser les llama Estados, y se llaman secuencialmente en el index.
